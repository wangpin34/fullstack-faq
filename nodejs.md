### Update nodejs or npm
* Mac/Linux/Window(with poweshell)
```bash
# Clean cache, usually it is not necessary
$ sudo npm cache clean -f
# n is tool for manage nodejs/npm versions
$ sudo npm install -g n 
$ sudo n stable 

```
